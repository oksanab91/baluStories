== Description ==

Travel Diaries is a modern, clean and easy to use WordPress theme suitable for traveller bloggers and all kind of bloggers. The design is responsive and therefore, each elements are displayed nicely in all kind of devices of various screen sizes. Theme theme has a custom home page with several features such as subscription form, recent posts, most popular articles, travel guide, featured on section with Call to Action (CTA) button for advertisement, and featured on section with Call to Action button for advertisement. It also has 4 custom widgets. The theme provides feature to write number of countries visited, last trip and next trip. The theme is translation ready.

== Installation ==
        
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Documentation ==
1. To set up the homepage you must first create a new page.
    * Go to Pages > Add New.
    * Enter the Page Title for the Page.
    * Set featured Image for the home page by clicking on Set Featured Image. The selected Image will then Appear on the Homepage below the Primary menu. The recommended size for the image is 1920*548 pixels.You do not require to enter anything in the content section for creating a home page.
    * Select the Homepage Template from the Page Attributes section
    * Click Publish.

2.Configuring the front Page display setting
    * Go to Appearance>Customize > Default Settings> Static Front Page.
    * Select A static page under Front Page displays
    * Choose the page you just created as Front page.
    * Click Save & Publish.

3. Header Info Section
    * Go to Appearance> Customize> Header Info Settings
    * Check Enable Header Info
    * Enter the required Labels and the respective content.
    * Click Save & Publish

4. Banner Settings
    * Go to Appearance> Customize>Home Page Settings> Banner Settings
    * Check Enable Banner Section to display a banner on the homepage.
    * Check Enable Banner Form to display subscription form on the homepage.
    * Check Enable Featured On Section to display various Logos.
    * Upload the required logos, Logo URLs, Featured On Link Text, and Featured On Link URL for the Featured On Section.
    * Click Save & Publish.

5. Recent Post Section
    * Go to Appearance>Customize> Home Page Settings>Recent Post Settings
    * Check Enable Recent Post Section to display recent posts from your blog on your homepage.
    * Enter Recent Post Title, Recent Post Content and Recent Post View All Text.
    * Click Save & Publish
    
6. Popular Article Setting
    * Go to Appearance>Customize> Home Page Settings>Popular Article Settings
    * Check Enable Popular Article Section to display selected posts from your blog on your homepage.
    * Enter Popular Article Title and Popular Article Content. Select Posts to be displayed on the homepage. Select either three or six posts to maintain the design.
    * Click Save & Publish

7. Client Settings
    * Go to Appearance>Customize> Home Page Settings>Client Settings
    * Check Enable Client Section to display it on the homepage.
    * Enter Clients Section Title, Logo URLs, Link Text and Link URL. Upload Logos by clicking on Select Image.
    * Click Save & Publish.

8. My Guide Settings
    * Go to Appearance>Customize> Home Page Settings>My guide Settings
    * Check Enable My Guide Section to display it on the homepage.
    * Enter My Guide Title and My Guide Content. Select the Posts to be displayed. Select either three or six posts to maintain the design.
    * Click Save & Publish.

9. Banner Widget
    * You need to first install and activate a plugin, Newsletter By Stefano Lissa, The Newsletter Team, for the Newsletter Subscription form to appear on the Banner Widget. To learn how to install a plugin, read Installing Plugin. Set up your Newsletter subscription form details as per http://www.thenewsletterplugin.com/plugins/newsletter/newsletter-documentation
    * Go to Appearance> Customize> Homepage Settings> Banner Widget
    * Click Add a Widget. There is a description of each included widget that you can choose from.
    * Click on the desired widget. For Newsletter subscription form, Click Newsletter Widget.
    * Click Save& Publish.
    
10. For more details documentation please visit http://raratheme.com/documentation/travel-diaries/
